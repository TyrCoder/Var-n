## üéØ Key Concepts

### Three Island Groups
- **üèùÔ∏è Luzon**: Northern Philippines
- **üèùÔ∏è Visayas**: Central Philippines  
- **üèùÔ∏è Mindanao**: Southern Philippines

### How It Works
1. Seller selects island in settings
2. Header shows island badge
3. When releasing order to riders
4. Only riders in that island appear
5. Seller assigns to best rider
